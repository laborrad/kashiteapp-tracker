# セキュリティポリシー

## 脆弱性の報告

kashiteappに関するセキュリティ上の問題を発見された場合は、以下の方法でご報告ください。

### 報告方法

**重大な脆弱性の場合:**
- **公開Issueには投稿しないでください**
- GitHub Security Advisoriesを使用: [こちらから非公開で報告](../../security/advisories/new)
- または、開発チームに直接ご連絡ください

**軽微な問題の場合:**
- 通常のIssueとして報告していただいて構いません
- ただし、具体的な攻撃方法は記載せず、概要のみをお願いします

## 対応について

- セキュリティ報告は優先的に確認します
- 重大度に応じて迅速に対応いたします
- 修正完了後、報告者にお知らせします

## 個人情報の取り扱い

このリポジトリは公開されています：
- **個人情報を投稿しないでください**（メールアドレス、電話番号、住所など）
- スクリーンショットに個人情報が含まれていないかご確認ください
- 必要に応じて画像にモザイクをかけてください

## 不正利用について

以下の行為は禁止されています：
- スパム投稿
- 無関係な宣伝・広告
- 他者への嫌がらせや誹謗中傷
- 不適切なコンテンツの投稿

違反を発見された場合は、開発チームにご報告ください。

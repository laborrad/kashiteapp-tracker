# kashiteapp Issue Tracker

このリポジトリは **kashiteapp** のIssue管理専用です。ソースコードは別のプライベートリポジトリで管理されています。

## 利用目的

- バグ報告
- 機能要望
- 質問・問い合わせ

## Issue作成方法

1. [Issues](../../issues)タブを開く
2. "New issue"をクリック
3. テンプレートを選択して入力

## 対応フロー

開発チームが確認後、適切なラベルを付与し対応します。重要度や優先度に応じて対応時期が決定されます。

---

**Note**: このリポジトリにはコードは含まれません。Issue管理のみを目的としています。

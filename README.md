# kashiteapp サポート窓口

**kashiteapp** をご利用いただきありがとうございます。

こちらは、アプリのバグ報告、機能要望、使い方の質問などを受け付ける専用の窓口です。

## 報告・お問い合わせ方法

1. 上部の [**Issues**](../../issues) タブをクリック
2. 右上の **New issue** ボタンをクリック
3. 内容に応じたテンプレートを選択
   - **バグ報告**: アプリの不具合を報告
   - **機能要望・改善提案**: こんな機能が欲しい
   - **質問・問い合わせ**: 使い方がわからない
4. フォームに沿って入力して送信

## 対応について

- ご報告いただいた内容は開発チームで確認し、順次対応いたします
- 重要度や影響範囲を考慮して優先順位を決定します
- 対応状況はIssueのコメントで随時お知らせします

## ご注意

- GitHubアカウントが必要です（無料で作成できます）
- 個人情報は記載しないようお願いいたします
- 不具合報告の際は、できるだけ詳しい情報をお願いします（再現手順、環境など）

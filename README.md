# kashiteapp サポート窓口

**kashiteapp** をご利用いただきありがとうございます。

こちらは、アプリの不具合連絡、改善アイデア、使い方の質問などを受け付ける専用の窓口です。
お気軽にご連絡ください。

> 💡 **ご利用には[GitHubアカウント](https://github.com/signup)が必要です（無料で作成できます）**

## 📝 連絡方法

### 1. [Issues](../../issues) ページを開く
上部の [**Issues**](../../issues) タブをクリックしてください。

### 2. **New issue** をクリック
右上の緑色のボタンを押すと、テンプレート選択画面が表示されます。

### 3. 内容に合わせてテンプレートを選択

#### 🔧 不具合連絡
アプリがうまく動かない、おかしいな？と思ったらこちら

**記入例:**
```
タイトル: ログイン画面でパスワードが入力できない

- どんな状況か: パスワード入力欄をタップしてもキーボードが出ない
- いつ・どこで: 今朝、ログイン画面で
- お使いの端末: iPhone 13
```

👉 [不具合連絡の例を見る](../../issues?q=is%3Aissue+label%3Abug+state%3Aclosed)

#### 💡 改善など
こうなったらもっと使いやすい、こんな機能があったらいいな、と思ったらこちら

**記入例:**
```
タイトル: 検索結果を日付順に並べ替えたい

- どんなこと: 検索結果を新しい順・古い順に並べ替えたい
- どんな時に: イベント検索で日付順に見たい時
```

👉 [改善提案の例を見る](../../issues?q=is%3Aissue+label%3Aenhancement+state%3Aclosed)

#### ❓ その他連絡
使い方の質問、感想、その他何でもお気軽にどうぞ

**記入例:**
```
タイトル: 通知の設定方法を教えてください

通知をオンにしたいのですが、設定画面のどこから変更できますか？
```

👉 [すべてのIssueを見る](../../issues)

### 4. 内容を記入して送信
テンプレートに沿って記入し、**Submit new issue** をクリックすれば完了です。

---

## 📋 対応の流れ

1. ご連絡いただいた内容を開発チームで確認
2. 必要に応じて追加質問をコメントでお聞きすることがあります
3. 対応状況や結果をIssueのコメントでお知らせします
4. 対応完了後、Issueをクローズします

重要度や影響範囲を考慮して優先順位を決定しますので、対応までお時間をいただく場合があります。

---

## ⚠️ ご注意

- **個人情報**（メールアドレス、電話番号など）は記載しないようお願いします
- スクリーンショットを添付する際は、個人情報が映り込んでいないかご確認ください

## 🙋 よくある質問

<details>
<summary>Q. GitHubアカウントを持っていません</summary>

[GitHub](https://github.com/signup) から無料で作成できます。メールアドレスがあれば数分で登録完了します。
</details>

<details>
<summary>Q. 急ぎの不具合はどうすればいいですか？</summary>

重大な不具合の場合は、Issueのタイトルに「【至急】」を付けてください。優先的に確認します。
</details>

<details>
<summary>Q. 自分の連絡を見返したい</summary>

[Issues](../../issues) ページで、検索ボックスに `is:issue author:@me` と入力すると、自分が作成したIssueだけが表示されます。
</details>
